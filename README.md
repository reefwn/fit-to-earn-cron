# Fit To Earn Cron

## Installation

```
yarn install
```

## Running the app

```
# development
yarn start

# watch mode
yarn start:dev

# production mode
yarn run start:prod
```

## Test

```
# unit tests
yarn test

# e2e tests
yarn test:e2e

# test coverage
yarn test:cov
```
